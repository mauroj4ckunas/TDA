
Para ejecutar el programa de la Parte 1 del TP 1 de Teoría de Algorítmos se debe usar el siguiente comando: 

> python *promocion.py* *ejemplo.txt*

Donde 'ejemplo.txt' debe cumplir con el siguiente formato:

~~~
    1,Cacho,8,3,4 
    2,Lucho,10,5 
    3,Suncho,15,1
    4,Pucho,40,1
    5,Tucho,25,2
    6,Fercho,9,
~~~
